# Flight Delay Analysis

## Project Overview

Data analytics project examining airline delays to identify major causes of disruption and compare airline performance.

## Research Questions

- What are the largest causes of flight delays?
- Which airlines experience the highest delay rates?
- How do delay patterns vary across carriers?

## Dataset

Large-scale U.S. airline operations and flight delay records.

## Methods

- SQL querying
- MongoDB aggregation
- Data cleaning
- Exploratory data analysis

## Key Findings

- Delay causes varied significantly across airlines.
- Certain carriers experienced consistently higher delay frequencies.
- Weather and carrier-related issues were major contributors to disruptions.
