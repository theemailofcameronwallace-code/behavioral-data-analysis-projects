# MLB Pitch-by-Pitch Data Analysis

## Overview
This project analyzes MLB pitch-by-pitch data to identify patterns in pitching strategy, player performance, and game outcomes. The analysis focuses on transforming large baseball datasets into clear insights using statistical exploration and visualization.

## Goals
- Explore trends in pitch selection and pitcher behavior
- Analyze relationships between pitch type, count, and outcomes
- Identify patterns in hitter performance against different pitches
- Practice large-scale sports data analysis and visualization

## Dataset
The dataset contains pitch-by-pitch information from MLB games including variables such as:
- Pitch type
- Pitch velocity
- Batter and pitcher IDs
- Count and game situation
- Pitch result (ball, strike, hit, etc.)

## Methods
The analysis includes:
- Data cleaning and preprocessing
- Exploratory data analysis
- Statistical summaries
- Data visualization to highlight trends and patterns

## Tools Used
- R
- tidyverse
- ggplot2

## Key Skills Demonstrated
- Data cleaning and wrangling
- Exploratory data analysis
- Sports analytics
- Data visualization
