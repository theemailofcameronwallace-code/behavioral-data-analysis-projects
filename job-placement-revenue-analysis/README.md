# Job Placement Revenue Analysis

## Project Overview

Business analytics project evaluating salary trends and recruiting opportunities across industries and experience levels.

## Business Context

A recruiting company earns revenue as a percentage of candidate salaries. This project identifies which industries and experience levels generate the strongest placement opportunities.

## Research Questions

- Which industries offer the highest salaries?
- Which experience levels generate the highest placement value?
- Which recruiting segments provide the greatest revenue opportunity?

## Methods

- Exploratory Data Analysis
- Salary benchmarking
- Industry comparisons
- Tableau dashboard development

## Key Findings

- Salary levels varied substantially across industries.
- Mid-career professionals represented strong placement opportunities.
- Data visualization helped identify high-value recruiting segments.
