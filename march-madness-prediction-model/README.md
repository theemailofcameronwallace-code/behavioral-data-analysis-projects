# March Madness Prediction Model

A statistical model that predicts NCAA March Madness tournament outcomes using seed-based win probabilities, historical upset rates, and controlled randomness.

## Goal
To simulate realistic tournament brackets by combining:
- Team seed differences
- Historical upset probabilities
- Random performance variance

## Method

The model calculates a probability that a lower seeded team wins based on historical upset data.

Example factors used:
- Seed matchup probabilities
- Strength difference between teams
- Randomized variance to simulate game unpredictability

## Tools Used
- Excel
- Random simulations

## Example Outcome

The model generates simulated tournament brackets and identifies:
- Likely first-round upsets
- Potential Cinderella teams
- Most probable Final Four combinations
