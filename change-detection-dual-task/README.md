# Change Detection Dual Task

## Project Overview
This project examines how working memory load influences performance on a change detection task.

## Research Question
How does adding a working memory task affect accuracy and reaction time in change detection?

## Dataset
Behavioral dataset including:
- 30+ participants
- ~1000 trials
- reaction time and accuracy measurements

## What I Did
- Helped design and run the behavioral paradigm
- Organized participant datasets
- Compared performance across conditions
- Interpreted working memory effects on performance

## Tools Used
Excel  
R  
PsychoPy  
Data cleaning  
Data visualization

## Key Takeaway
Participants often showed slower responses or reduced accuracy when performing change detection while holding items in working memory.
