# Visual Search Dual Task

## Project Overview
This project examines how working memory load influences performance during a visual search task. Participants complete a visual search while simultaneously maintaining information in working memory.

## Research Question
Does maintaining information in working memory slow visual search reaction times or reduce accuracy?

## Dataset
Behavioral dataset including:
- visual search trials
- working memory dual-task conditions
- reaction time measurements
- accuracy comparisons across conditions

## What I Did
- Supported development and implementation of the experiment
- Organized participant behavioral datasets
- Compared performance across experimental conditions
- Interpreted reaction time and accuracy patterns across tasks

## Tools Used
Excel  
R  
PsychoPy  
Data cleaning  
Data visualization  

## Key Takeaway
The project investigates how increased cognitive load interferes with efficient visual search and decision speed.
